다크 모드와 라이트 모드를 모두 지원해 밤낮 어떤 환경에서도 편안하게 사용할 수 있습니다. 스마트폰, 태블릿, PC 등 모든 기기에 최적화된 반응형 디자인을 적용했습니다.

동물상 테스트
AI(MobileNet v2)가 사진을 분석해 강아지상·고양이상·믹스상을 판별합니다.

주요 특징

  • 완전 무료: 모든 서비스를 무료로 이용할 수 있습니다. 회원가입 불필요.
  • 개인정보 보호: 동물상 테스트 이미지는 서버로 전송되지 않고 브라우저 내에서만 처리됩니다.
  • 다크/라이트 모드: 우측 상단 버튼으로 언제든지 테마를 전환할 수 있습니다.
  • 모바일 최적화: 스마트폰, 태블릿, PC 어디서나 편리하게 이용 가능합니다.
  • 빠른 로딩: Cloudflare Pages CDN을 통해 전 세계 어디서나 빠르게 접속됩니다.

사용 기술

본 사이트는 아래 기술을 활용해 제작되었습니다.

  • HTML5 / CSS3 / JavaScript: 별도 프레임워크 없이 순수 웹 표준 기술로 구현
  • TensorFlow.js + MobileNet v2: 동물상 테스트 AI 분석 (브라우저 내 실행)
  • Fisher-Yates Shuffle: 통계적으로 편향 없는 무작위 번호 추첨 알고리즘
  • CSS Custom Properties: 다크/라이트 모드 테마 전환
  • Cloudflare Pages: 정적 사이트 호스팅 및 글로벌 CDN
  • Formspree: 서버리스 제휴 문의 폼 처리
  • Disqus: 댓글 서비스

로또 번호 추첨기 소개

로또 번호 추첨기는 대한민국 로또 6/45의 번호를 무작위로 생성해주는 도구입니다. 피셔-예이츠(Fisher-Yates) 셔플 알고리즘을 사용하여 1~45 사이의 숫자 중 중복 없이 7개(메인 6개 + 보너스 1개)를 선택합니다.

번호는 구간별로 색상이 구분됩니다. 1~10번은 노란색, 11~20번은 파란색, 21~30번은 빨간색, 31~40번은 초록색, 41~45번은 보라색으로 표시되며 보너스 번호는 회색으로 구분됩니다. 추첨 내역 기능으로 같은 세션 내에서 생성한 번호를 한눈에 확인할 수 있습니다.

동물상 테스트 소개

동물상 테스트는 구글이 개발한 MobileNet v2 딥러닝 모델을 활용해 업로드한 사진을 분석하고, 강아지상·고양이상·믹스상 중 하나로 분류하는 재미있는 서비스입니다.

MobileNet v2는 ImageNet의 1,000가지 카테고리를 학습한 경량 고성능 이미지 분류 모델입니다. 사진에서 감지된 특징들을 강아지·고양이 관련 키워드로 분류하여 각 유형의 유사도를 백분율로 계산합니다. 모든 처리는 브라우저 내에서만 이루어지며, 업로드한 이미지는 외부 서버로 전송되지 않아 개인정보가 안전하게 보호됩니다.

본 테스트는 오락 목적의 재미 콘텐츠이며 과학적·의학적 근거를 보장하지 않습니다.
